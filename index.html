CC Fest Creative AI / Machine Learning Camp

Learning Machines: Text, Images, Video

A tool ecosystem for investigating how generative systems write, imagine, and move. These tools are built for creative learning, classroom adaptation, and critical reflection.

Repeated question

What is the machine actually doing?

Tool principle

Every tool should make something invisible visible.

Pilot ready path

Start here for a live test session

Use this path to run a 60-90 minute pilot with the published tools, collect feedback, and identify launch blockers before expanding the ecosystem.
