Session link sheet

One canonical link per tool, grouped by session in run-of-show order. During a live session these are the links that get pasted into Zoom chat — everyone lands on the same page, one click deep. Use the copy button to grab a session's links as a ready-to-paste block.

Running a session live? The Run console pairs each of these links with its run-of-show timing and a launch button — the whole session from one tab.
