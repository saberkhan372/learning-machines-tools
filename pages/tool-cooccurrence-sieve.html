Relational Co-Occurrence Sieve

Two real facts can still compose a lie. After confirming each entity exists, this sieve pulls the subject's real Wikipedia page and checks that every other name, date, and place actually appears on it — catching 'Lincoln used an iPhone in 1865.'

§ B · How to investigate it

run it like an experiment, not a toy
01 · Predict

Before the audit

Predict whether the entities truly belong together.

guess: iPhone doesn't fit 1865
02 · Change one thing

Swap one relationship

Keep the entities real; change how they relate.

true date → false date
03 · Compare evidence

Read both levels

Passes existence but fails co-occurrence — or both?

L1 pass, L2 fail = relational lie
04 · Name it

Name the lie

Say where it lives: a fake entity, or a fake relationship.

Type 2 = false relationship
