CFG Scale Visualizer

Move classifier-free guidance from loose to useful to brittle. The tool shows why more prompt pressure can improve alignment at first, then create stiff, repeated, over-literal artifacts.

§ A · What it makes visible

prompt pressure
Fig. 01

Unconditioned habit

At very low guidance, the system leans on broad learned defaults. The image can look plausible while ignoring prompt details.

Fig. 02

Useful guidance

In the middle range, the prompt direction steers subject, setting, and style while preserving enough variation to look natural.

Fig. 03

Over-correction

At high guidance, the prompt gets amplified so hard that the simulated image becomes crowded, harsh, and artifact-prone.

§ B · How to investigate it

run it like an experiment
01 · Predict

Before moving the slider

Ask learners what they think happens when guidance increases. Better image? More literal image? Stranger image?

prediction: "higher CFG will follow the prompt better"
02 · Compare presets

Use 1, 7, 14, 30

Move through the four presets and name what changes: prompt match, leftover defaults, and artifact pressure.

CFG 7: useful · CFG 30: brittle
03 · Change prompt family

Does the pattern hold?

Switch scenarios and repeat the same slider movement. The mechanism should stay visible even as the prompt changes.

same dial, different scene
04 · Make a claim

Not just "high is bad"

Write a precise claim about the tradeoff: where prompt alignment improves, and where over-guidance starts to damage the image.

"CFG is useful until it begins amplifying artifacts"

§ C · Debrief questions

after the investigation
Where did the prompt first become clearly visible?
At what point did the image start feeling over-forced?
What did the low-CFG version fill in from model habit?
Why is guidance not the same thing as quality, truth, or ethics?
