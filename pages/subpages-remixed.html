Mapping

Each skin is matched to the page type it flatters — a tool earns the terminal, a long read earns the editorial zine, a session opener earns the poster. The mapping isn't fixed; any skin can wrap any type. Compare against the production pages: tool · doc · session — those ride the riskier identity data-skin layer; this sheet is the proofing reference for how far each skin can be pushed.
