Run the No-AI pathway without making it secondary.

Use this page when you need the tool list, session language, and activity sequence. The participant-facing page stays shorter and focuses on the promise: direct generation is optional, but participation is not reduced.

A.

Facilitator Language

Say this early and plainly: "You can complete this camp without generating new AI content. You can analyze examples, run simulations, build a critique, make a classroom activity, or create a consent checklist. That work has equal standing."

C.

A Simple No-AI Session Plan

  1. Choose one question.
    For example: what does the model assume, what breaks over time, or when does confidence exceed evidence?
  2. Pick frozen evidence.
    Use prompt packs, screenshots, pre-generated images, frame strips, or simulated tool outputs.
  3. Predict before looking closely.
    Write what you expect the default, failure, or answer to be.
  4. Annotate what appears.
    Name the visible evidence: wording, image detail, missing context, unstable object, or unsupported claim.
  5. Write a claim.
    Use the frame: "This matters because..." and "The evidence is..."
  6. Turn it into a shareable artifact.
    Create a critique, checklist, classroom activity, model card, poster, or facilitation plan.
