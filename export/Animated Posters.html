Getting the MP4s

GIFs (in export/gifs/) are finished — use them directly. For MP4: install ffmpeg, then from the project root run bash export/build-mp4.sh. Files land in export/mp4/, each at the framerate that matches its GIF.

Output is 417×540 (and 540² / 304×540 for the social pair) — ideal for web & social. Need native 850×1100? Use the standalone poster HTML with a headless recorder (timecut / Puppeteer); that's the only route this environment can't finish itself.
