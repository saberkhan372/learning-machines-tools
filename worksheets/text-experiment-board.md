# Text Experiment Board

Use this with the Tokenizer + Temperature Visualizer, ELIZA Simulator, or any text model investigation.

Participation options:

- Use: generate or test text directly.
- Observe / Critique: analyze pre-generated text examples.
- Teach / Design: adapt the prediction game or ELIZA comparison.
- Build / Code: make a small text explainer or prompt test.
- Critical / No-AI: critique a chatbot interface, recap policy, or classroom use case.

## Reflection

- Human: what did you choose, frame, select, revise, or reject?
- Machine: what did the system predict, complete, or distort?
- System: what defaults, interface choices, or training patterns shaped the result?
- Ethics: who might be copied, misrepresented, erased, stereotyped, or excluded?
- Pedagogy: how would this need to change for students or community learners?
- Did the model understand the story, or imitate patterns of stories?
- What did I control through language?
- What did the model add without being asked?
- What would I need to revise as a human creator?
- What ethical concern did this raise?
