# Text Experiment Board

Use this with the Tokenizer + Temperature Visualizer, ELIZA Simulator, or any text model investigation.

Participation options:

- Use: generate or test text directly.
- Observe / Critique: analyze pre-generated text examples.
- Teach / Design: adapt the prediction game or ELIZA comparison.
- Build / Code: make a small text explainer or prompt test.
- Critical / No-AI: critique a chatbot interface, recap policy, or classroom use case.

## How to run it

This board is your lab notebook — it doesn't talk to a model. Open any chat model in another window, run the prompts there, and log what happened here.

1. Pick one small topic (a story, an apology note, a lesson intro). You'll reuse it in every row.
2. **A — Baseline:** a plain prompt on your topic. Run it and note what came back.
3. **B — Change one thing:** the same prompt with exactly one deliberate change — genre, audience, tone, or a constraint. Run it in a fresh chat so answer A can't influence it.
4. **C — Respond to something:** stay in the chat this time. Push on whatever surprised you in the output — a wrong fact, an odd assumption, an unasked-for detail — and log how the model handles the challenge.
5. **D — Push further (optional):** exaggerate the change from B, or stack two changes, until the output breaks or gets strange.

"Prompt / genre frame" means the framing you hand the model: your topic plus a genre, audience, tone, or format — "write about rain" vs. "write a weather report about rain."

Worked example (one A → B pair): A — "Write a four-sentence story about a family dinner." B — the same prompt "as a nature-documentary voiceover." What changed: same dinner, new voice — B added a hushed narrator and suspense nobody asked for. What I learned: one genre word swaps the whole vocabulary; the model carries a genre's defaults with it.

## Reflection

- Human: what did you choose, frame, select, revise, or reject?
- Machine: what did the system predict, complete, or distort?
- System: what defaults, interface choices, or training patterns shaped the result?
- Ethics: who might be copied, misrepresented, erased, stereotyped, or excluded?
- Pedagogy: how would this need to change for students or community learners?
- Did the model understand the story, or imitate patterns of stories?
- What did I control through language?
- What did the model add without being asked?
- What would I need to revise as a human creator?
- What ethical concern did this raise?
