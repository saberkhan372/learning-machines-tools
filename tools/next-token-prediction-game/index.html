Next-Token Prediction Game

Before opening a language model, humans guess the next word in a sentence. This tool collects those guesses from a Zoom chat paste and shows the room's distribution beside the model's predicted probabilities.

The gap between the two distributions is the discussion. Every tool should make something invisible visible.
