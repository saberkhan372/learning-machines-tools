Metronome Frame-Scrubber

Video adds a dimension text and stills never had: time. A video model must track pixel-vectors across frames so an object doesn't mutate from one moment to the next. Widen the temporal window and motion locks together; shrink it to one frame and the subject melts into spatiotemporal drift.

Big idea: coherence over time is attention, not magic — narrow the window and the subject forgets itself.

The backlit light-box

An animator drawing a flipping pancake never draws blind. They set sheet #2 on a glowing light-box over sheet #1 and trace the existing edges, so the pancake shifts only a millimeter per frame. That backlit stack of past sheets is temporal attention. Switch it off — attend to one frame — and every drawing starts from scratch: the pancake bulges, twists, and dissolves. Engineers call it spatiotemporal drift.
